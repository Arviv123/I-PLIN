# Iplan MCP Server

שרת MCP לגישה לנתוני מינהל התכנון הישראלי

## התקנה

```bash
npm install
npm start
```

השרת יפעל על פורט 10000

## נקודות קצה

- `/` - בדיקת בריאות
- `/sse` - נקודת הקצה של MCP

## כלים זמינים

- חיפוש תכניות
- פרטי תכנית ספציפית  
- חיפוש לפי מיקום
- הגבלות בנייה
- נתוני תשתיות
- אתרי שימור
